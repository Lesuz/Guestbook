This project has four routes and is made with Node and express:

  • Frontpage
  • Guestbook
  • New Message
  • AJAX message

The guestbook displays messages which are saved on a local json file. It will display the message, and the username and country of the one who left the message. A new message can be added to this json file by filling in the form on the New Message -page, which can be found in the navigation on top of this page.

Requirements for this project:

  • 4 routes present
  • Homepage has to have actual content
  • Guestbook loads and formats data
  • Forms check for basic errors (no empties allowed)
  • Saving data to a JSON file
  • Publish app in Heroku
